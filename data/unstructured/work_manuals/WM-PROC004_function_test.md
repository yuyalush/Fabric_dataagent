# WM-PROC004 機能検査 標準作業手順書
**文書番号**: WM-PROC004-REV3  
**工程**: PROC-004 機能検査  
**対象製品**: IVC-3000、IVC-5000、SMD-500、SMD-200、PLC-2000  
**制定日**: 2024-05-01 / 改訂日: 2025-11-01  
**承認者**: 品質保証部長 鈴木 恵美

---

## 1. 適用範囲

組立完了品の出荷前機能検査全般に適用する。本工程は  
PROC-004（機能検査）と PROC-005（最終検査）の入口工程を兼ねる。  
検査員は機能検査資格（FT-Level1 以上）を保有すること。

---

| 設備 ID | 設備名 | 用途 | 校正周期 |
|---------|--------|------|---------|

---

## 3. 検査フロー概要

```
外観検査 → 絶縁耐圧試験 → 電源投入試験 → 基本動作確認
    → 通信機能試験 → 負荷試験 → 電力損失測定 → 最終合否判定
```

---

## 4. 検査項目別手順

### 4.1 外観検査（所要時間: 約 5 分）

1. 品番・シリアル番号ラベルが正しく貼付されていることを確認する。
2. 筐体の傷・凹み・塗装不良がないことを確認する（QS-PROD001 / QS-PROD002 参照）。
3. コネクタ・端子台が全て嵌合・固定されていることを確認する。
4. チェックシート（FC-004-01）に記録する。

### 4.2 絶縁耐圧試験（所要時間: 約 10 分）

| 試験項目 | 試験電圧 | 試験時間 | 合格基準 |
|---------|---------|---------|---------|
| 主回路 – 筐体間 | AC 1,500V | 1 秒 | 漏れ電流 ≤ 20mA |
| 制御回路 – 筐体間 | AC 500V | 1 秒 | 漏れ電流 ≤ 5mA |
| 絶縁抵抗（主回路）| DC 1,000V | — | ≥ 100MΩ |

> ⚠️ IVC-5000 は主回路 AC 2,000V にて試験すること。

### 4.3 電源投入試験（所要時間: 約 5 分）

1. EQ-401 を定格電圧（IVC-3000: AC 200V）に設定する。
2. 電流計の指示値を確認する（突入電流が 30A を超えていないこと）。
3. ステータス LED の点灯パターンを確認する。
   - スタンバイ時: 緑点灯（1Hz 点滅）
   - エラー時: 赤点滅（エラーコードをLCD表示）
4. 異臭・煙の発生がないことを確認する（30 秒間）。

### 4.4 基本動作確認（所要時間: 約 15 分）

IVC-3000 の場合:
1. PCコントロールソフト "IVC-Configurator v5.2" を起動する。
2. COM ポートに接続し、設定パラメータをデフォルト値で書き込む。
3. 空転（無負荷）にて周波数 10Hz/30Hz/50Hz の各ステップで動作確認する。
4. 加速時間 (0→50Hz: 5秒)・減速時間 (50Hz→0: 5秒) が設定値通りであることをオシロで確認する。
5. FabTest SW-401 に測定値を記録し、合否判定を行う。

### 4.5 通信機能試験（所要時間: 約 10 分）

1. RS-485 テスター（EQ-406）を接続する。
2. Modbus RTU コマンドを送信し、レスポンスを確認する。
   - Function Code 03 (読取): 保持レジスタ 40001-40010 を読み取り、全て応答すること
   - Function Code 06 (書込): 40001 に 0x1234 を書込み、反映確認
3. 通信エラーカウンタが 0 であることを確認する。

### 4.6 負荷試験（所要時間: 約 20 分）

1. EQ-404（可変負荷）を 25% → 50% → 75% → 100% の順に増加させる。
2. 各段階で 3 分間保持し、以下を記録する。
   - 出力電圧・電流
   - IGBT 温度（EQ-405 ロガーで記録）
   - 制御基板温度
3. **IGBT 温度が 85°C 以下**であることを確認する。（IVC-3000 定格時）
4. 100% 負荷で 10 分間連続運転後、パラメータに変化がないことを確認する。

### 4.7 電力損失測定（所要時間: 約 10 分）

| 項目 | 基準値 | 測定値記録欄 |
|------|--------|------------|
| 無負荷損失 | ≤ 50W | ___ W |
| 50% 負荷効率 | ≥ 95% | ___ % |
| 100% 負荷効率 | ≥ 96% | ___ % |

---

## 5. 不合格時の処置

| 不合格項目 | 即時措置 | 報告先 |
|-----------|---------|--------|
| 絶縁耐圧不合格 | 作業停止・隔離 | 品質保証部・製造部長 |
| IGBT 温度超過（>85°C） | 放熱経路確認（放熱グリス・フィン密着） | 生産技術部 |
| 通信エラー | 配線確認（WM-PROC003 照合）→ マイコン交換 | 品質保証部 |
| 効率基準未達 | IGBT 部品ロット確認 → 不適合ロット即時停止 | 購買部・品質保証部 |

---

## 6. 検査記録の保管

- 検査結果は FabTest SW-401 に保存し、製造番号と紐付ける。
- チェックシートは5年間保管する（品質保証部が管理）。
