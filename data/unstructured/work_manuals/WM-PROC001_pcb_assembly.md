---

## 1. 目的

本手順書は、プリント基板（PCB）への電子部品実装作業を標準化し、品質の均一化と作業安全性の確保を目的とします。

---

## 2. 適用範囲

インバータ制御装置・サーボモータードライバー・PLC ユニットの制御基板・電力基板を対象とする。

---

## 3. 安全注意事項

- **静電気対策（必須）**: 作業前に必ずアース付きリストバンドを装着してください。ESD（静電気放電）により IGBT モジュールや制御 IC が破損します
- **はんだの取り扱い**: 鉛フリーはんだ（Sn-Ag-Cu 系）は溶融温度が 217〜220℃と高め。やけどに注意
- **換気**: はんだフュームが発生するため、排気設備が稼働していることを確認してください
- **保護具**: 保護メガネ・作業手袋を着用してください

---

## 4. 必要工具・設備

| 工具/設備 | 仕様 | 備考 |
|---|---|---|
| はんだごて | 温度調節型（350〜400℃） | 鉛フリー対応チップ使用 |
| はんだ | Sn96.5-Ag3-Cu0.5、0.8mm径 | JIS Z 3282 準拠 |
| フラックス | 水溶性フラックス | 実装後の洗浄が必要 |

| リストバンド（アース付） | 1MΩ抵抗入り | ESD対策 |
| はんだ吸取線 | 2.5mm幅 | 修正作業用 |

## 5. 作業手順

### 5.1 事前準備（約5分）

1. 作業台の清掃・静電気対策（アース確認）
2. 作業指示書で**製品型番・基板品番・ロット番号**を確認する
3. 部材棚から当該基板・電子部品を受け取り、**員数を確認**する（部材員数チェックシートに記載）
4. はんだごての電源を入れ、**設定温度 370℃**に調整する（5分以上ウォームアップ）

### 5.2 SMD 部品の実装（約15分）

1. 専用ジグに基板をセットする
2. **チップコンデンサ・抵抗**から実装する（小型部品を先に）
3. 部品をピンセットでパッドに配置し、**片側をはんだ付け**（仮固定）
4. 位置ズレがないことを確認してから**反対側をはんだ付け**
5. ルーペで**はんだブリッジがないか**確認する

> ⚠️ **NG 事例**: はんだが隣接パッドにブリッジしている場合は、吸取線で吸い取り、再実装する

### 5.3 スルーホール部品の実装（約15分）

1. **電解コンデンサ**はリードを曲げて基板に差し込む。**極性に注意**（＋側のマーキングを確認）
2. **コネクタ**のリードを差し込み、**仮固定**（ラジオペンチで軽く折り曲げ）
3. 裏面からはんだ付けを行う。はんだ量は**リード径の2倍程度**
4. はんだが冷えたら（5秒後）リード余長をニッパーでカットする

### 5.4 IGBT / パワー半導体の実装（約10分）

1. IGBT モジュールは**熱伝導グリス**を塗布してヒートシンクに取り付ける
   - グリス塗布量: 0.5g（薄く均一に）
   - 締付トルク: M4 ネジ 2.0 N·m
2. IGBT のリードを基板に差し込み、はんだ付け
3. **ゲート抵抗の値**を確認する（製品別に異なる）

### 5.5 外観検査・完了（約5分）

1. **フラックスの洗浄**（水溶性フラックスは純水で洗浄後、エアブローで乾燥）
2. ルーペ・顕微鏡でははんだボール・ブリッジ・部品欠落がないか確認
3. 作業記録シートに実装完了時刻・作業者 ID を記入する
4. 次工程（筐体組立）の作業台に移送する

---

## 6. 品質チェックポイント

| チェック項目 | 判定基準 |
|---|---|
| はんだブリッジ | 隣接パッド間の接触なし |
| はんだ不足 | フィレットが確認できること |
| 部品向き・極性 | 基板シルク印字と一致 |
| 部品欠落 | 部品員数チェックシートと一致 |
| フラックス残渣 | 洗浄後に残渣がないこと |
| IGBT グリス塗布 | 均等塗布・気泡なし |

---

## 7. 不良発生時の対応

1. 不良品に**NG タグ**を貼付し、不良品トレーに移動
2. **不良内容**をはんだ付け不良記録票に記載（日付・製品型番・不良内容・作業者）
3. 班長（田中）に**即時報告**する
4. 班長判断で**手直し（リワーク）** or **廃棄**を決定
