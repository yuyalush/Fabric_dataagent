## 3. 工程別品質基準

### 3.1 PROC-001 基板実装

IVC-3000（QS-PROD001）と共通。QS-PROD001 § 3.1 を参照のこと。

追加項目:
- SMD-500 専用チップ（モーション制御 IC: MC-501H）のハンダ接合を X 線検査で全数確認すること。  
  合格基準: ボイド率 ≤ 15%

### 3.2 PROC-002 筐体組立

| 部位 | 管理項目 | 規格値 | 測定方法 |
|------|---------|--------|---------|
| フロントパネル取付ネジ | トルク | 0.5±0.1 N·m | トルクドライバー（TL-301） |
| エンコーダコネクタ | 挿入力 | 5〜15N | プッシュプルゲージ |
| 冷却ファンマウント | 振動 | 10Hz〜55Hz で異常音なし | |
| 筐体ねじれ精度 | 対角寸法差 | ≤ 0.5mm | ノギス |

### 3.3 PROC-003 配線作業

| 配線 | 管理項目 | 規格値 |
|------|---------|--------|
| エンコーダケーブル | シールド接続 | 基板側 SHD に接続、逆側フリー |
| EtherCAT ケーブル | インピーダンス | 100Ω ±10% |
| 電源入力 | 絶縁抵抗 | DC 500V で ≥ 10MΩ |
| モータ出力 UVW | 相間抵抗差 | ≤ 1% |

### 3.4 PROC-004 機能検査

| 検査項目 | 規格値 | 測定方法 |
|---------|--------|---------|
| 絶縁耐圧（主回路-筐体） | AC 1,000V / 1秒 / 漏れ電流 ≤ 20mA | EQ-403 |
| スタンバイ消費電力 | ≤ 10W | EQ-401 電力計 |
| 最大速度応答性 | 設定値の ±2% 以内 | EQ-404 + エンコーダフィードバック |
| 位置決め精度 | ±1 パルス（エンコーダ 16384 ppr） | EQ-404 |
| EtherCAT 通信遅延 | ≤ 1ms (1000BASE-T) | EQ-406 |
| 発熱（IGBT 温度）| ≤ 80°C（定格 100% 負荷 10 分後） | EQ-405 |

### 3.5 PROC-005 最終検査（抜取検査）

AQL 2.5 にて抜取検査を実施する。  
ロットサイズ 26〜50 台の場合: サンプル数 8 台、合格判定数 0、不合格判定数 1。

| 抜取検査項目 | 確認内容 |
|------------|---------|
| 製品ラベル | 品番・シリアル番号・製造年月の正確性 |
| 附属品 | 取扱説明書・保証書・付属ケーブルの同梱 |
| 動作確認 | 電源 ON → 3 分間動作 → 異常なし |
| 外観 | キズ・汚染なし |

## 5. 不合格品の処置

| 不合格カテゴリ | 発見工程 | 処置 |
|-------------|---------|------|
| MC-501H ボイド率 > 15% | PROC-001 | 全数廃棄・サプライヤー通知 |
| IGBT 温度 > 80°C | PROC-004 | 放熱グリス再塗布 → 再試験。2回再発で廃棄。 |
| 位置決め精度不合格 | PROC-004 | ゲインパラメータ再調整後に再試験（最大2回）。改善なし → 廃棄。 |
| 外観傷（最終検査） | PROC-005 | 傷の深さ・幅を評価。軽微→出荷判定基準書(DS-012)を適用。 |
